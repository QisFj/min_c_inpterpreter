基本运算:
- \+
    
    二目运算
    
    重载 4 个:
    + int (int,int)
    + real (int,real)
    + real (real,int)
    + real (real,real)

- \-
    二目运算
    
    重载 4 个:
    + int (int,int)
    + real (int,real)
    + real (real,int)
    + real (real,real)

- \*
    二目运算
    
    重载 4 个:
    + int (int,int)
    + real (int,real)
    + real (real,int)
    + real (real,real)

- /
    二目运算
    
    重载 4 个:
    + int (int,int)
    + real (int,real)
    + real (real,int)
    + real (real,real)

- %
    二目运算
    
    重载 1 个:
    + int (int,int)

- $

    二目运算

    重载 1 个:
    + string (string,string)

- \#

    单目运算

    重载 1 个:
    + string(string)
